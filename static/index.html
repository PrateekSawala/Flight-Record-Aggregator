Flight ticket upgrader processes the flight records. Then writes the successful entries to the passed record file, The records that fail the validation, get stored in the failed record file. So, that someone can look at them and fix the problem. Both files get stored inside the uploads folder of the application.

After the upload, If the files get processed successfully, The passed and failed records of the uploaded file can also be found over these link - Processed files.

Instructions For uploading a file:

1. Click on the "Choose Files" button to upload a file.

2. Click on the "Upload files" button to send the uploaded files.

Note: Only CSV file formats are allowed for the file upload, File Upload Size limit is 5 MB
